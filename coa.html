    • Performance Issues: Designing for performance, Multicore, Mics, GPGPU.

    • Virtual Memory: Concept, Segmentation and Paging, Page replacement policies. LRU,FIFO

    • Buses: Types of Buses ,Bus Arbitration, BUS standards
    • I/O Interface, I/O channels, I/O modules and IO processor, Types of data transfer techniques: Programmed I/O, Interrupt driven I/O and DMA.
    • Introduction to parallel processing concepts ,Flynn's classifications,pipeline processing ,Pipeline stages, Hazards
