Operating System

Syllabus

  1. Introduction to Operating systems

    1. Introduction to operating system, Functions of Operating systems, Evolution of Operating Systems, Booting.
    2. Processes and Interrupts, User mode and kernel mode, Kernel architectures: Monolithic, Micro-kernel,RTOS, Mobile OS.
    3. Process System Calls

  2. Processes and Threads.

    1. Life cycle of a process, context switching, Process Control Block, Inter-process communication (IPC).
      2.
    2. Process Scheduling: FCFS, RR, SJF , Priority , Comparison of different scheduling policies.
    3. Threads and Thread management

  3. Process Synchronization

    1. Race conditions and Locks, Conditional Variables
    2. Mutual Exclusion- Hardware and Software approaches, Semaphores, Monitors.
    3. Examples of synchronization problems: Readers Writers problem, Dining Philosophers problem, Producer Consumer problem.

  4. Memory Management

    1. Basics of Memory Management, Address Space of a Process
    2. Paging, TLB cache, Page table design
    3. Working set model, Thrashing, Demand Paging, Page Replacement Algorithms: FIFO,LRU,Optimal.
    4. Frame Allocation Policies, Kernel Memory Allocation
    5. Segmentation

  5. File Management

    1. Files and File Systems, Layers of Abstraction in File Systems
    2. Design Choices for File and Directory Metadata
    3. File System Consistency
    4. Buffer Cache and Memory-mapped Files
    5. Unix and Windows file systems

  6. I/O Management

    1. Kernel I/O subsystem
    2. Communication and Data Transfer with I/O Devices
    3. Secondary Storage Disks, Disk Scheduling and disk scheduling algorithms : FCFS , SSTF , SCAN , CSCAN , LOOK , Disk cache, RAID. <

  7. Explore operating system functions in real time operating system, mobile operating system, distributed operating system and cloud environment
